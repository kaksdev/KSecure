# KSecure
Simple encrypt and decrypt


## Getting Started

```
# Normal Encrypt And Decrypt
string encryptedText = KSecure.Normal.Encrypt(plainText, password);
string plainText = KSecure.Normal.Decrypt(encryptedText, password);
```


### Prerequisities

.NET Framework 4
